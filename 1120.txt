Practical 17: Chess Board Pattern
Aim: Generate chessboard layout.
Logic: Cell color depends on row-column parity.
Program Code:
<?php
echo "<table border='1' width='200' height='200'>";
for ($i = 1; $i <= 8; $i++) {
    echo "<tr>";
    for ($j = 1; $j <= 8; $j++) {
        $color = (($i + $j) % 2 == 0) ? 'white' : 'black';
        echo "<td bgcolor='$color'></td>";
    }
    echo "</tr>";
}
echo "</table>";
?>
Output: Chess board displayed

Practical 19: Sum of Digits
Aim: Calculate sum of digits.
Logic: Loop extracts digits sequentially.
Program Code:
<?php
function sumDigits($n) {
    $sum = 0;
    while ($n > 0) {
        $sum += $n % 10;
        $n = intdiv($n, 10);
    }
    return $sum;
}
echo sumDigits(123);
?>
Output: 6
Practical 20: Factorial of Number
Aim: Calculate factorial using function.
Logic: Multiplicative loop execution.
Program Code:
<?php
function factorial($n) {
    $fact = 1;
    for ($i = 1; $i <= $n; $i++) {
        $fact *= $i;
    }
    return $fact;
}
echo factorial(5);
?>
Output: 120

Practical 21: Check Prime Number
Aim: Check whether a number is prime.
Logic: Divisibility tested up to square root.
Program Code:
<?php
$n = 7;
$isPrime = true;
if ($n <= 1) $isPrime = false;
for ($i = 2; $i <= sqrt($n); $i++) {
    if ($n % $i == 0) {
        $isPrime = false;
        break;
    }
}
echo $isPrime ? 'Prime Number' : 'Not Prime';
?>
Output: Prime Number
